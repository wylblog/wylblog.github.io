Ubuntu安装安装 IntelliJ 全家桶

下面以idea作为例子

一.下载IntlliJ IDEA

IntlliJ IDEA的版本自己选择:

官网:IntelliJ IDEA – the Leading Java and Kotlin IDE (jetbrains.com)

专业版下载:

社区版下载:

二. 下载好后解压到指定路径：

# 解压到
sudo tar -xvf ideaIU-2024.2.3.tar.gz -C /opt/
# 更改权限组
cd /opt/
sudo chown wyl:wyl -R idea-IU-242.23339.11/

三. 打开解压后的文件位置

注：这时候其实可以打开idea了 进入到 IDEA文件夹的bin目录 终端输入

cd /opt/idea-IU-242.23339.11/bin/
./idea.sh

四. 添加IDEA快捷图标：

在/usr/share/applications/文件夹里面创建一个.desktop文件

sudo vim /usr/share/applications/idea.desktop

添加以下内容,路径改成自己的:

[Desktop Entry]
Name=IntelliJ IDEA
Comment=IntelliJ IDEA
# 安装地址下的idea.sh
Exec=/opt/idea-IU-242.23339.11/bin/idea
# 图标路径
Icon=/opt/idea-IU-242.23339.11/bin/idea.png
Terminal=false
Type=Application
Categories=Developer;

:wq保存退出

五. 启动测试

1.打开菜单栏

2.找到idea图标

3.单机启动即可启动
